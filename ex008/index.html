Texto Grande e Texto Pequeno

Texto Grande

Estamos criando um texto grande com a tag <big>.

Essa tag já está descontinuada

Texto Pequeno

Estamos criando um texto pequeno com a tag <small>.

Texto Excluído, Texto Inserido, Texto Sublinhado e Texto Sobrescrito

Podemos marcar um texto como Excluído usando a tag <del>.

Podemos marcar um texto Inserido usando a tag <ins>.

Existe também o texto Sublinhado usando a tag <u>. Essa TAG é não semântica!

Existe também o texto Sobrescrito usando a tag <sup>. Para inserir coisas do tipo x20+3

Existe também o texto Subescrito usando a tag <sub>. Para inserir coisas do tipo H2O
